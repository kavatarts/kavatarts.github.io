# Kavat Art Studio Website

A beautiful, modern website for Kavat Art Studio by Keerthi Tajaswini, a professional drawing teacher and artist. This website showcases various art forms and provides information about art classes.

## Features

### 🎨 Beautiful Design
- Modern, elegant design with smooth animations
- Gradient backgrounds and artistic color schemes
- Professional typography using Playfair Display and Poppins fonts
- Responsive layout that works on all devices

### 📱 Fully Responsive
- Mobile-first design approach
- Hamburger menu for mobile devices
- Optimized layouts for tablets and desktops
- Touch-friendly interface

### ✨ Interactive Elements
- Smooth scroll navigation
- Animated counters for statistics
- Hover effects on cards and images
- Parallax scrolling effects
- Cursor trail animation
- Fade-in animations on scroll

### 🎭 Art Forms Covered
1. **Pencil Drawing** - Fundamentals, shading, portraits, still life
2. **Watercolor Painting** - Color theory, wet techniques, landscapes
3. **Charcoal Drawing** - Tonal values, blending, figure drawing
4. **Acrylic Painting** - Brush techniques, layering, abstract art
5. **Colored Pencils** - Blending, realistic textures, botanical art
6. **Sketching** - Quick drawing, urban sketching, travel journaling

## How to Use

1. **Open the website:**
   - Simply open `index.html` in any modern web browser
   - Or use a local server for best experience

2. **Navigate:**
   - Click on navigation links to jump to sections
   - Scroll naturally to explore content
   - Use the contact form to send inquiries

3. **Customize:**
   - Update contact information in `index.html`
   - Modify colors in CSS variables in `styles.css`
   - Add real images by replacing placeholder elements
   - Customize art form descriptions as needed

## Customization Guide

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #9b59b6;    /* Main purple color */
    --secondary-color: #e74c3c;  /* Red accent */
    --accent-color: #3498db;     /* Blue accent */
    --dark-color: #2c3e50;       /* Dark text */
    --light-color: #ecf0f1;      /* Light background */
}
```

### Adding Real Images
Replace the placeholder elements in `index.html`:
- About section: Replace the SVG with `<img src="path/to/image.jpg" alt="Keerthi Tejaswini">`
- Gallery: Replace `.gallery-placeholder` divs with actual images

### Updating Contact Information
Edit the contact section in `index.html`:
- Email address
- Phone number
- Location
- Class hours
